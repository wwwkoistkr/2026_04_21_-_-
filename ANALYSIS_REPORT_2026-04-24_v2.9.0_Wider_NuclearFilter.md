### 📐 요구 1+2: 화면폭 확대 (노란색 박스 + 전체 너비)

**현재 상태 (v2.8.0)**:

**v2.9.0 변경**:
| 항목 | v2.8.0 | v2.9.0 | 변화 |
|---|---|---|---|
| 메일 전체 max-width | 680px | **920px** | +35% |
| 데스크톱에서 좌우 여백 | 크다 | **거의 없음 (꽉 참)** | ✅ |
| 모바일 (≤480px) 너비 | 100% | **100% + 좌우 padding 12px** | 동일 |
| 노란색 박스 padding | 14px 18px | **20px 24px** | +43% |
| 노란색 박스 border-left | 4px | **6px** | +50% |
| 노란색 박스 font-size | 24px | **26px** | +8% |
| 노란색 박스 margin | 12px | **16px** | +33% |

**기술 구현**:
```css
/* HTML_TEMPLATE */
<table width="920" style="max-width:920px;width:100%;">
/* 기존 680px → 920px */

/* 노란색 투자 시사점 박스 확대 */
.card-insight {
  padding: 20px 24px !important;
  border-left-width: 6px !important;
  font-size: 26px !important;
  margin: 16px 0 !important;
}

/* 태블릿 (481~900px)에서 중간 너비 */
@media only screen and (min-width: 481px) and (max-width: 900px) {
  table[width="920"] { max-width: 100% !important; }
}
```

**예상 효과**:
- 데스크톱 1920px 화면: 좌우 여백 **(1920-920)/2 = 500px → 가독성 우수**
- 태블릿 768px: **자동 100% 축소 → 좌우 여백 없음**
- 모바일 iPhone 390px: **자동 100% + 12px padding → 글자 안 잘림**
